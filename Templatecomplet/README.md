# Jardins Provence Paysage — Technical SEO Pack

Pack généré pour un site vitrine lead-gen local : SEO local, services, zones, réalisations, blog et conversion.

## Fichiers inclus

- schema.global.jsonld
- schema.home.jsonld
- schema.service.jsonld
- schema.local.jsonld
- schema.blog-post.jsonld
- schema.breadcrumbs.jsonld
- robots.txt
- manifest.json
- humans.txt
- llms.txt
- sitemap-index.xml
- sitemaps/sitemap-pages.xml
- sitemaps/sitemap-services.xml
- sitemaps/sitemap-local.xml
- sitemaps/sitemap-realisations.xml
- sitemaps/sitemap-blog.xml

## À vérifier avant production

- Téléphone réel
- Email réel
- Adresse réelle
- SIRET / TVA réels
- Coordonnées GPS
- Horaires
- Avis clients vérifiables et affichés sur le site
- Assets images réellement présents
- Query params bloqués dans robots.txt
- Restrictions d’eau à maintenir à jour si article publié

## Recommandation

Pour la production, injecter les JSON-LD page par page plutôt qu’un seul gros @graph partout.
